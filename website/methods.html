Methodologies

To provide a clearer understanding of the visual methodologies employed in this project, each visualization tool used will be presented on this page. This will assist the audience in comprehending the data and the insights gained from each visualization. The tools used, including choropleth maps, bubble plots, and box plots, were selected based on their ability to communicate the relationship between selected variables in a clear and concise manner. By presenting each tool individually, the audience will be able to appreciate the different aspects of the data that were revealed through the visualizations. Moreover, this section will show the final dataset we use.

Combine_data.csv: Dataset Overview

Combine_data.csv was combined by three raw datasets, which are WorldHappiness_Corruption_2015_2020.csv , life_ladder.xls, and world_bank.csv

Combine_data.csv: Variables Overview

Variable Name Type Description
Year float The year, from 2008 to 2021
Continent string Continent Name(Asia, Europe, North America, South America, Africa, Oceania)
Country string Country Name, total 143 countries
Country Code string Country Code from ISO 3166-1 alpha-3
Happiness Score float Happiness Score with range from 0 to 10
Log GDP float Purchasing Power Parity (PPP) in constant 2017 international dollars. Uses the natural log of GDP per capita, as this form fits the data significantly better than GDP per capita.
Social Support float The average of binary responses (0=no, 1=yes) from the question: 'If you were in trouble, do you have relatives or friends you can count on to help you whenever you need them, or not?'
Health float Life expectancy based on the health
Freedom float The average of binary responses (0=no, 1=yes) from the question: 'Are you satisfied or dissatisfied with your freedom to choose what you do with your life?'
Generosity float The average of binary responses (0=no, 1=yes) from the question: 'Have you donated money to a charity in the past month?'
Corruption float The average of binary answers to two questions: 'Is corruption widespread throughout the government in this country or not?' and 'Is corruption widespread within businesses in this country or not?'
Positive Affect float The average of previous-day affect measures for laughter, enjoyment, and doing or learning something interesting
Negative Affect float The average of previous-day affect measures for worry, sadness, and anger
Confidence float The average of binary responses (0=no, 1=yes) from the question: 'Are you confident in your government?'
CO2 Emissions float Total CO2 emissions with kt unit
Population float Total population
Unemployment float The percentage of the unemployment rate of total labor force
Education float Total number of children not attending primary school
Adjusted Net National Income float Adjusted net national income in current U.S. dollars

Data Visualization Methodologies

Name Description Code
Figure 1: Choropleth Map The choropleth map was created using Plotly in Python. The Scattergeo() function was utilized to map the world and annotate emojis for each country. The slider located at the bottom of the graph can adjust the year variable to indicate happiness scores for different years. The metric used to measure the happiness score of each country is the "Happiness Score" variable. The histogram distribution of the “Happiness Score” indicates that approximately one-third of the variable is around 4.5, while two-thirds are around 6. The metric is used to label the countries with emojis. Link
Figure 2: Boxplot To observe the distribution of the happiness score, a Boxplot was created using Plotly. The "go.Box()" method was combined with "add_trace()" to add each continent as a different trace to the figure. The "visible" method was then set to true to display all the traces together. Additionally, to compare the Happiness Scores of different continents in the same year, the 'boxmode = group' parameter was used to display the different continents together. Link
Figure 3 & Figure 4: Bar Plot and Line Plot

To compare the happiness scores of each continent and 12 countries, we chose to draw bar and line plots in Python using Altair. In Figure 3, the Y-axis displays the 6 continents, and the X-axis represents the average happiness score for each continent. In the bar plot shown in Figure 4, the Y-axis displays the average happiness score of the 12 selected countries, and the X-axis displays the names of the 12 countries. The line plot in Figure 4 displays the happiness score on the Y-axis, the year on the X-axis, and the colors in the plot represent the different countries.

Furthermore, the bar and line plots are combined in one graph. The bar plot in both Figure 3 and Figure 4 is arranged by average happiness scores from highest to lowest. Additionally, in Figure 4, clicking on the bar of any country in the bar plot displays the corresponding country's happiness scores over the years in the line plot. The high-to-low ranking helps readers better compare the average happiness scores of different continents or countries.

 Link
Figure 5: Heatmap To show the correlation among different variables, the "df.corr()" method was used to calculate the correlation values of every numerical variable. The "go.Heatmap()" method was used to visualize the correlation coefficient matrix, with red representing positive correlation and blue representing negative correlation. The "colorscale" parameter was used to set the color scheme. The depth of color indicates the strength of the correlation, with deeper colors indicating higher correlation values. In addition to the color, the "hoverongaps" parameter was set to True to display the correlation coefficient value when hovering over a cell. The "hovertemplate" parameter was also set to show the correlation coefficient value and the names of the two variables. Link
Plot 6: Line Plot

This line plot was created using Plotly and Python to illustrate the changes in Positive Affect and Negative Affect from 2008 to 2021 across six continents. The plot was generated using the "go.Scatter()" method. It uses the same color for each continent through the "legendgroup" parameter in "go.Scatter()". However, to differentiate between Positive and Negative Affect, the Negative Affect line style was changed to a dashed line generated by the "dash='dash'" parameter.

Additionally, to provide more information about the Positive or Negative Affect values, year, and continent, the "hovertemplate" parameter was used. This allows for customization of the hover tooltip to display relevant information when hovering over a data point.

Link
Plot 7: Scatter Plot To investigate whether freedom to make life choices affects happiness, we created a scatterplot with a trendline and a boxplot with X and Y axes. The X and Y axes in Figure 7 represent the values of two variables (happiness score and freedom), and the colors represent different continents. The scatterplot is used to observe the relationship between freedom to make life choices and happiness among the six continents, while the box plot is used to examine the minimum, maximum, and median values of happiness scores and freedom for the six continents. We generated the plot using the Plotly library in Python, using the "px.scatter()" function to create the scatterplot and "marginal" and "trendline" to add the box and line. Link
Plot 8: Bubble Plot To show the relationship between the average happiness index and GDP per capita for each country, a bubble chart was chosen as the visualization tool. The chart consists of bubbles, where the position of the bubble on the X and Y axes represents the values of the two variables, and the size of the bubble represents the value of the third variable. Bubble charts are a good choice for visualizing data sets with multiple variables and comparing their relationships. Additionally, when viewers hover over each point, they can easily see information about the country, population, and happiness index that the point represents. The bubble chart also includes a time slider, which allows viewers to observe changes in the relationships between the variables over time. Link
Figure 9: Parallel Coordinate Chart To examine the relationship between the Happiness score and the other four variables simultaneously, we created a parallel coordinate graph. The x-axis of the graph represents the four quantitative attributes: Social support, Healthy life expectancy at birth, Confidence in the national government, Corruption, and Happiness score. The y-axis represents the values of each dimension. Each line represents each continent for each year. The chart is colored using a color scale that is defined based on the Happiness score of each country. The graph is generated using the Plotly library in Python, with the "go.Figure()" function, and the "go.Parcoords" function is used to create the parallel coordinate graph. Link
Figure 10: Radar Plot & Bar Chart These two plots were created using Plotly in Python. To enhance visualization, the five thetas in the radar plot were normalized using the min-max feature scaling method. This method brings all values into the range of [0, 1]. To apply this method, each value needs to be normalized by subtracting the minimum value and dividing by the difference between the maximum and minimum values. After normalization, all components are able to share the same polar axis. In the next step, twelve traces were created and grouped by continent, resulting in six groups. Each trace represents one subplot of the radar plot or bar chart with a single continent. Finally, a subplot was created, and all traces were added to it. Link
